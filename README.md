# body-mass-index-calculator
BMI Converter is a simple web application that helps you calculate your Body Mass Index (BMI) based on your height and weight. It provides you with an easy way to track whether your weight falls within the healthy range according to your height.


## Built With 
- HTML5 (JSX)
- Styled Components
- Mobile First Approach
- Media Queries
- React
- Javascipt
- useState Hook
